# Nextern

# Project Vision
Nextern is a mobile application designed to provide learners with easy access to learning programs while giving administrators
tools to manage programs, assignments, and learner activities through a simple and user-friendly interface.

## Objectives

- Provide learners with access to available programs.
- Enable learners to monitor their progress.
- Allow administrators to manage learning programs and assignments.
- Create a mobile app that helps learners and administrators.

## Navigation Flow

### Learner Flow
1. **Login / Sign Up** → User logs in or creates an account as a Learner
2. **Dashboard (Home)** → Displays course completion progress, current module, and saved favorite programs
3. **Browse** → Explore available programs/internships, search and filter by category
4. **Tasks (Assignments & Certificates)** → View and submit assignments, track submission status, access certificates
5. **Profile** → Manage account details

Bottom navigation bar (Home, Browse, Tasks, Profile) allows learners to move freely between these sections at any time.

### Administrator Flow
1. **Admin Sign Up / Login** → Register or log in with organization details
2. **Admin Dashboard** → View system overview: active users, average program completion, quick actions, and recent submissions
3. **Admin Monitoring (Tasks)** → Track learner progress and review submitted feedback
4. **Users / Programs** → Manage users and learning programs
5. **Settings** → Configure system/account settings

Bottom navigation bar (Dashboard, Users, Updates, Settings) mirrors the learner navigation pattern for consistency.
