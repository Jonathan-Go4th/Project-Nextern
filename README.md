# Nextern

# Project Vision
Nextern is a mobile application designed to provide learners with easy access to learning programs while giving administrators
tools to manage programs, assignments, and learner activities through a simple and user-friendly interface.

## Objectives

- Provide learners with access to available programs.
- Enable learners to monitor their progress.
- Allow administrators to manage learning programs and assignments.
- Create a mobile app that helps learners and administrators.
